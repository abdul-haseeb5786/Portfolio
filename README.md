# 🚀 Abdul Haseeb - Full Stack Developer & AI Automation Engineer

Welcome to my portfolio! This project is a modern, high-performance web application built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**. It showcases my journey as a developer, highlighting projects ranging from SaaS platforms to AI-driven automation systems.

---

## ✨ Key Features

- **🌍 Internationalization (i18n):** Full support for English, Arabic, and Spanish, ensuring a global reach.
- **📱 Progressive Web App (PWA):** Installable on mobile and desktop devices with offline support.
- **🎨 Modern UI/UX:** Built with **Tailwind CSS 4**, **Framer Motion** for smooth animations, and **Hero UI** components.
- **🌓 Dark Mode:** Seamless theme switching with `next-themes`.
- **📊 Dynamic Case Studies:** Detailed project views with metrics, technical challenges, and implementation results.
- **⚡ Performance Optimized:** High Lighthouse scores and fast transitions using Next.js App Router.

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abdul-haseeb5786/Portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   cd my-portfolio-next
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:3000` to see the magic!
